## Decisions Made

- The picker remains presentation-only convenience. Direct AJAX and drift-panel semantics still use the existing server handler contract.
- Wider eligibility beyond administrator-role users is deferred because enforcing it now would change authorization scope and violate the plan boundary.
- User option text includes display name plus login for disambiguation without adding new data sources.

- **Verification:** `composer analyse` passed.
