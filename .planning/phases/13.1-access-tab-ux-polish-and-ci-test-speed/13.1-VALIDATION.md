## Test Infrastructure

| Property | Value |
|----------|-------|
| **Framework** | PHPUnit 9.6 (unit + integration); Playwright 1.58.2 (E2E) |
| **Config file** | `phpunit.xml.dist`; `playwright.config.ts` |
| **Quick run command** | `./vendor/bin/phpunit tests/Unit/AdminTest.php` (~0.3s) |
| **Full suite command** | `composer test:unit` (~4s, 810 tests) + `composer analyse` (PHPStan L6) |
| **E2E spec** | `npm run test:e2e -- tests/e2e/specs/access-grant.spec.ts` |
| **Estimated runtime** | unit ~4s; static ~30s; targeted E2E ~4–5min |

CIS track is CI-workflow configuration — validated in GitHub Actions, not by a local framework.

---

## Sampling Rate

- **After every task commit (ACC track):** `./vendor/bin/phpunit tests/Unit/AdminTest.php`
- **After every plan wave:** `composer test:unit` + `composer analyse`
- **Before `/gsd:verify-work`:** full unit suite green; `access-grant.spec.ts` green; CIS workflow changes green in actual CI
- **Max feedback latency:** ~4s (unit); CI roundtrip for CIS track
- **CIS track:** validated by CI run on the PR — no per-commit local sample applies (config only)
